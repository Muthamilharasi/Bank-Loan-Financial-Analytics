# 🏦 Bank Loan Financial Analytics 

## Problem
Banks lacked a unified view to identify high-risk loan segments,
default patterns, and NPA exposure across customer demographics.

## Approach
- Cleaned 10,000+ loan records using Python (Pandas)
- Created risk categories, DTI ratio, and default flags
- Built interactive Power BI dashboard with 10 DAX measures
- Applied global slicers for dynamic filtering

## Key Insights
1. Very High Risk borrowers default at 3× the rate of Low Risk
2. Debt consolidation is the most common loan purpose (40%+ of loans)  
3. Low Income borrowers show highest default concentration
4. NPA exposure concentrated in Large loan size category

## Tools Used
Python · Pandas · Matplotlib · Numpy 

## Files
- loan_cleaning.py    → Data cleaning script
- loan_eda.py         → EDA + chart generation  
- cleaned_loan_data.csv → Cleaned dataset
- loan_eda_charts.png → EDA visualizations
- Financial_Dashboard.pbix → Power BI dashboard
